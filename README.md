## 📌 Featured Projects

Here are some highlights from my portfolio, showcasing my skills and passion for development.

### 1. **EduTrack – Classroom & Student Management Portal**
  <img src="https://raw.githubusercontent.com/KartikkxCode/KartikkxCode/main/assets/edutrack.png" alt="EduTrack Project Thumbnail" width="100%" style="margin-bottom:10px;"/>
-   **Technology Stack:** Java, Spring Boot, MongoDB, REST APIs
-   **Description:** Developed and deployed a robust full-stack system for comprehensive classroom and student record management. Implemented efficient CRUD operations via RESTful APIs and a NoSQL database, complemented by a user-friendly Java-based client interface.
-   **Explore:** [EduTrack on GitHub](https://github.com/KartikkxCode/edutrack-portal)

### 2. **Customised Virtual File System (CVFS)**
  <img src="https://raw.githubusercontent.com/KartikkxCode/KartikkxCode/main/assets/cvfs.png" alt="CVFS Project Thumbnail" width="100%" style="margin-bottom:10px;"/>
-   **Technology Stack:** C Programming, OS Internals
-   **Description:** Engineered a simulation of a Linux-like file system, complete with a custom shell and implementations of core system calls (`open`, `read`, `write`, `lseek`, `delete`). This project provided profound insights into **inodes, file tables, and operating system internals**.
-   **Explore:** [CVFS on GitHub](https://github.com/KartikkxCode/CVFS)

### 3. **File Packer & Unpacker with Encryption**
  <img src="https://raw.githubusercontent.com/KartikkxCode/KartikkxCode/main/assets/filepacker.png" alt="File Packer Project Thumbnail" width="100%" style="margin-bottom:10px;"/>
-   **Technology Stack:** Java, Encryption
-   **Description:** Built a versatile file archiving utility featuring robust encryption and decryption capabilities. The tool ensures metadata preservation during packing/unpacking and offers an intuitive graphical user interface (GUI) developed using Java Swing.
-   **Explore:** [File Packer on GitHub](https://github.com/KartikkxCode/File-Packer-Unpacker)

### 4. **Generalised Data Structures Library**
  <img src="https://raw.githubusercontent.com/KartikkxCode/KartikkxCode/main/assets/datastructures.png" alt="Data Structures Project Thumbnail" width="100%" style="margin-bottom:10px;"/>
-   **Technology Stack:** C++
-   **Description:** Developed a highly reusable and generic library of fundamental linear and non-linear data structures. This includes template-based implementations of stacks, queues, linked lists, Binary Search Trees (BST), and various searching & sorting algorithms, ensuring data type independence.
-   **Explore:** [Data Structures Library on GitHub](https://github.com/KartikkxCode/Generalised_DS)
